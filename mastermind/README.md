# Mastermind

## How to play

The object of the game is to deduce the randomly generated secret code within twelve turns, based on the hints provided.  When the game is started, the computer generates a code consisting of four colors. Valid colors include:

- (R)ed
- (G)reen
- (B)lue
- (C)yan
- (M)agenta
- (Y)ellow

The same color can appear multiple times in the secret code.

On your turn, you make a guess by entering four letters corresponding the colors listed above.  The computer will then provide you with up to four hints based on your guess, represented by red and yellow asterisks.  A yellow asterisk indicates that you have guessed one of the colors correctly, but that it is in the wrong position.  A red asterisk indicates that you have correctly guessed both the color and position.

Good luck, and have fun!
